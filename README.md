### 🍀 선뜻 (Fine Meaning)
- 봉사, 후원 및 커뮤니티 웹 서비스
- kh정보교육원 파이널 프로젝트 (인원: 5명)
- 개발기간: 2023.06 ~ 2023.07

### 🙋‍♀️ 사용자
- 흩어져있는 봉사, 후원 정보들을 한데 모아 보고자 하는 유저
- 진행하고 있는 후원에 대한 투명성을 보장하는 사이트를 원하는 유저
- 회원들과의 자유로운 소통을 원하는 유저

### 🪄 기술 스택
- Front-End
  - HTML5, CSS3, JavaScript, jQuery
- Back-End
  - Java 11, Servlet, SQL, Ajax
  - Spring Boot, Oracle 19c
- Tools
  - STS4, VS Code

### 📂 주요 기능
- 선뜻
  - 선행인물과 최신 뉴스 조회
  - 후원 내역 공개 게시판
- 봉사
  - 참여할 수 있는 봉사 조회 및 신청
- 후원
  - 후원할 수 있는 모금함 조회 및 기부
- 커뮤니티
  - 공지사항 게시판
  - 회원간 자유 소통 게시판
- 유저 (본인 담당)
  - 로그인 및 회원가입
  - 카카오 로그인
  - 아이디·비밀번호 찾기
